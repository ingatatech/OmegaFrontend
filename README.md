# OMEGA SIR Ltd Website

This is the official website for OMEGA SIR Ltd, built with Next.js and Tailwind CSS.

## Tech Stack
- Next.js (App Router, TypeScript)
- Tailwind CSS (with custom company colors)

## Brand Colors
- Primary: #1760b0
- Secondary: #f1b30e

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure
- `/app` - Main app directory (pages, layouts)
- `/public` - Static assets (images, icons)
- `/components` - Reusable React components (to be created)

## Customization
- Update `tailwind.config.js` to change or extend the color palette.
- Add your own images to `/public` (e.g., team photos, project images).
